🚀 SalesIQ — Deployment Guide

Follow these steps to get a live URL. Total time: ~10 minutes.

Step 1 → 2
Backend → Render.com
Python + FastAPI + Database
Handles AI analysis, file uploads
Free tier available
Step 3 → 4
Frontend → Vercel
React + Vite UI
You already have an account ✓
Free tier available
1
Push code to GitHub
One-time setup — takes 2 minutes
1
Create a new repository on GitHub

Go to github.com → click the green "New" button → name it salesiq → click "Create repository"

2
Open a terminal in the ai-sales-agent folder and run:
git init git add . git commit -m "Initial SalesIQ commit" git branch -M main git remote add origin https://github.com/YOUR-USERNAME/salesiq.git git push -u origin main
2
Deploy Backend to Render.com
This gives you the API URL that Vercel will talk to
1
Create a free account at render.com

Sign up at render.com → choose "Sign up with GitHub" (easiest)

2
Create a new Web Service

Click New + → Web Service → Connect your GitHub → choose the salesiq repo

3
Fill in these settings on the Render form:
Name: salesiq-backend Runtime: Python 3 Build Command: pip install -r backend/requirements.txt Start Command: uvicorn backend.main:app --host 0.0.0.0 --port $PORT Plan: Free
4
Add Environment Variable

Scroll down to Environment Variables → click "Add Variable"

Key: LLM_PROVIDER Value: mock
💡 Note: Using "mock" means the app works instantly without any OpenAI or Anthropic API keys. You can change this later to "openai" or "claude" when you're ready.
5
Click "Create Web Service" and wait ~3 minutes

Render will build and deploy. When done, you'll see a URL like:
https://salesiq-backend.onrender.com — copy this URL, you'll need it next!

3
Deploy Frontend to Vercel
You already have a Vercel account — this is fast
1
Go to vercel.com → click "Add New Project"

Import from GitHub → choose the salesiq repo

2
Vercel will auto-detect the settings from vercel.json

You should see: Build Command = cd frontend && npm install && npm run build

3
Add Environment Variable (IMPORTANT)

Before clicking Deploy, go to Environment Variables section and add:

Name: VITE_API_URL Value: https://salesiq-backend.onrender.com ← (paste your Render URL from Step 2)
4
Click "Deploy" and wait ~2 minutes

Vercel builds the React app. When done, you get a live URL like:
https://salesiq.vercel.app

4
Test Your Live App
Visit your Vercel URL and try the demo
✓
Open your Vercel URL in a browser

You should see the SalesIQ dashboard.

✓
Upload a test audio/video file

Since LLM_PROVIDER=mock, it will instantly analyze the call with demo data — no real transcription needed.

✓
Watch the real-time pipeline run

You'll see the 7-step pipeline progress bar live via WebSocket.

🎉 Your SalesIQ app is live!
